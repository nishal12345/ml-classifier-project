# Titanic ML Classifier

A simple machine learning project to classify Titanic passengers using Logistic Regression, Random Forest, and XGBoost.

## Features
- Data preprocessing (label encoding, handling missing values)
- Training and comparing 3 classifiers
- Evaluation with Accuracy, F1 Score, Confusion Matrix
